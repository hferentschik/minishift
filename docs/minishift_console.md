## minishift console

Opens/displays the OpenShift console URL for your local cluster

### Synopsis


Opens/displays the OpenShift console URL for your local cluster

```
minishift console
```

### Options

```
      --url   Display the OpenShift console in the CLI instead of opening it in the default browser
```
